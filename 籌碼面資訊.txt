1. 融資融券比（多空比）
融資融券比 = 融資餘額 / 融券餘額

常見判斷標準：

>10：市場過度樂觀，多方主導，可能有回檔風險

5~10：市場偏多，股價可能穩步上漲

2~5：市場趨於平衡

<2：市場偏空，可能有軋空機會

2. 融資使用率
融資使用率 = 融資餘額 / 可用融資額度

常見判斷標準：

>80%：過度槓桿，市場可能過熱，容易回檔

50~80%：市場活躍，多方占優勢

<50%：市場風險較低，但也可能代表觀望氣氛濃厚

3. 融券使用率
融券使用率 = 融券餘額 / 可用融券額度

常見判斷標準：

>80%：市場對個股看空情緒濃厚，容易出現「軋空」

50~80%：市場偏空，但尚未進入極端情況

<50%：市場看空力道不強，短期內股價可能較為平穩

4. 融資增減幅
短線熱度指標 = 當日融資變化額 / 前一日融資餘額

常見判斷標準：

>10%：融資大幅增加，市場過熱，短線有回調風險

5~10%：市場資金積極流入，股價可能有上升動能

<5%：資金流入穩定，市場健康

負值（減少）：市場可能進入觀望或修正

5. 融券增減幅
空方力道指標 = 當日融券變化額 / 前一日融券餘額

常見判斷標準：

>10%：市場對個股看空情緒大增

5~10%：市場有一定空方壓力

<5%：市場空方動能較弱

負值（減少）：可能有軋空行情，或市場開始回補空單

6. 軋空力道
軋空力道 =（5日均融券餘額 - 當日融券餘額）/ 5日均融券餘額

常見判斷標準：

>20%：有明顯軋空行情，股價可能快速拉升

10~20%：可能有部分軋空力道

<10%：空單回補不明顯

7.squeezeForce (擠壓力道) 的功能與計算方法
📌 1. 功能
squeezeForce 是用來衡量市場多空力量的變化，特別關注 融資（多方資金）和 融券（空方資金）之間的資金流向。
它的目標是找出當前市場是否存在 軋空（short squeeze） 或 多殺多（long squeeze） 的可能性。

當 squeezeForce 為正數 → 代表融資增長較快，市場多方力量較強，可能推動股價上漲。

當 squeezeForce 為負數 → 代表融券增長較快，市場空方力量較強，股價可能面臨下跌壓力。

當 squeezeForce 接近 0 → 表示多空資金變化相對均衡，市場趨於平穩。
2.計算
squeezeForce = 融資增減幅 (%) - 融券增減幅 (%)

---------------------------------------------

1. 主力動向（主力買賣超）
計算方式
主力買賣超 = (特定大戶買進金額 - 特定大戶賣出金額) / 總成交金額 × 100%

一般會選擇「前15大或前20大券商」的買賣金額來代表主力行為

判斷標準
>5%：主力積極買進，可能有上漲動能

0~5%：主力買超不明顯

<0%（負值）：主力賣超，可能有下跌風險

補充： 如果某股票主力買超高、但股價沒有明顯上漲，可能表示主力在吸籌（布局）；相反，若股價已大幅上漲但主力大幅賣超，可能是主力在出貨。

2. 散戶動向（融資變化）
散戶通常使用 融資 進行交易，因此可以透過 融資餘額 的變化來觀察散戶動向。

計算方式
散戶動向 = 當日融資變化 / 總成交量 × 100%

或直接觀察 融資餘額的增減趨勢

判斷標準
融資大增（>5%）：散戶瘋狂進場，可能是市場過熱訊號

融資穩定增長（1~5%）：市場偏多，股價可能穩步上漲

融資減少（負值）：散戶退場，可能是主力吸籌或市場轉弱

📌 特別提醒：
如果融資大增但股價沒漲，可能是主力利用散戶資金來出貨（誘多）；反之，融資減少但股價沒跌，可能是主力吸籌的機會。

3. 籌碼集中度（大戶持股比例）
這是分析籌碼是否集中在少數大戶手中的指標，反映個股是否具備「主力控盤」的特性。

計算方式
籌碼集中度 =（前10大、前15大、前20大股東持股數）/ 總股本 × 100%

判斷標準
>80%：超高集中，極可能是主力控盤股（小心主力出貨）

50~80%：籌碼相對集中，較容易有趨勢行情

20~50%：籌碼分散，股價容易受市場消息影響

<20%：籌碼極度分散，通常屬於散戶主導股，股價震盪大

補充：
如果籌碼集中度持續上升，代表大戶正在布局（可能是主力進場）；相反，集中度下降，可能代表主力正在減碼（出貨）。

4. 大戶進出
這是衡量市場上大戶（法人、主力）的資金流向，可以透過「大戶持股變化」或「大額交易」來觀察。

計算方式
大戶持股變化 =（大戶持股比例當前 - 大戶持股比例前一日）

大額交易比率 =（單筆交易超過100張的交易數量）/ 總成交量 × 100%

判斷標準
大戶持股增加：大戶正在進場，股價可能有上升機會

大戶持股減少：大戶可能在撤退，股價可能走弱

大額交易比率 > 30%：可能是主力進出頻繁的股票

📌 特別提醒：

大戶持股增加 + 股價上漲 → 可能是「主力吸籌」，可以關注

大戶持股減少 + 股價上漲 → 可能是「主力出貨」，要小心
